## 介绍

高效脚手架一枚，欢迎使用~
如果报错的话 请提个 issues~
另外服务端用的是 egg 搭建 集成 socket,redis（功能还在完善~）代码（ https://github.com/Murrayee/m-server ）

## start

* yarn install （尽量用 yarn 安装 ，npm 有可能会出问题 ）
* yarn run ios (android)
